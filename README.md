# AgenTerra

This project is a learning implementation based on [agno](https://github.com/agno-agi/agno), using Microsoft's Agent Framework.

## Security

This project implements comprehensive security scanning in the CI/CD pipeline:

- **Dependency Review**: Automatically scans pull requests for vulnerable dependencies
- **CodeQL Analysis**: Static code analysis for security vulnerabilities
- **Dependabot**: Automated dependency updates for NuGet packages and GitHub Actions

For detailed information about security scanning, see [Security Scanning Documentation](research/SECURITY_SCANNING.md).
